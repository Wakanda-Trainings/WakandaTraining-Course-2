Day 2 - jQuery and Event Delegation





The more event handlers you define, the more memory you use, which can end up decreasing performance and making the UI sluggish.



Event delegation can help prevent this and we can use jQuery to implement it.

What is Event Delegation?



Do not attach event handlers directly to every element you want to listen to.


Instead, attach a single event handler to a parent element of those elements and listen for events on it's children.
