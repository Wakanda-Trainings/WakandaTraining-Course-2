Day 2 - Wakanda Widget and Page Events





The Browser publishes events to allow developers access to interesting moments of the browser and web page.



  • The page loads
  • The user clicks on a button
  • The user tabs out of an input field

Wakanda Widget and Page Events


addListener()



Page Loads

WAF.addListener("document", "onLoad", function(event) {}, "WAF");


User clicks the button

WAF.addListener("button1", "click", function(event) {}, "WAF");


Wakanda Widget and Page Events


addListener()



We will name our function to be executed when the button is clicked

button1.click = function button1_click (event)
{
	// Add your code here
};

Now we create the listener

WAF.addListener("button1", "click", button1.click, "WAF");

We can remove the listener later if we want

$$('button1').removeListener("click", button1.click);
