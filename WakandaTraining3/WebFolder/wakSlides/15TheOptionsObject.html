Wakanda UI Coding


The Options Object





In the Wakanda Application Framework (WAF) API, calls to built-in methods usually involve sending requests to the server and waiting for a response.



Therefore, most client-side API methods should be called in asynchronous mode.



To execute a request in asynchronous mode, you have to pass at least one callback function.

The Options Object





For any WAF method that sends a request to the server you can pass an options object as a parameter.



You pass the callback function(s) as properties of this options object.

The Options Object





waf.sources.person.query("lastName = :1", queryVar + "*",
{
   onSuccess: function(event) {
      vcount = event.entityCollection.length;		
   },
   onError: function(error) {

   }
}
);
