Day 2 - Web Components





Benefits of Web Components:



  1. Reuse interface elements and functionality throughout your project.
  2. Reduce the number of DOM elements on a Page.
  3. Share interface elements and code between projects and developers.
  4. Modularized and organized code base that is less prone to errors.

Scripting Web Components





Widgets within the Web Component have IDs that prepended with the Component Widget's ID.



Example: A Button in a web component has an id myButton. The component widget has an id of component1


When the page is loaded the DOM ID of the button is component1_myButton

Tools To Help You Create Reusable Components





getHtmlId() returns a DOM reference to a local widget.



getHtmlObj() returns a jQuery reference to a local widget.
