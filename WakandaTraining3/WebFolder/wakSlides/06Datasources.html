Day 2 - Datasources




The Cornerstone of Wakanda Client-Side Development

Datasources


Javascript Object Implemented by the WAF




  • Provides Bundle of Services to Developer to Ease Development
  • Dispatcher Between Data Objects and Widgets
  • Multiple Widgets Can Subscribe To The Same Datasource

Types Of Datasources


Datasources do not contain the data they provide.

It relies on other constructs to house the data.



  1. Datastore class
  2. Relation Attribute
  3. Array
  4. Variable
  5. Object

Datasources On A List


Datastore Class    -    Relation Attribute    -    Array



  • Keeps Pointer To Current Element
  • Promotes Current Element For Easy Access
  • Provides Methods to Move Through Collection
  • Loads Data Asynchronously - Without Code
