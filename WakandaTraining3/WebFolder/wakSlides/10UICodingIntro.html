Widget API





With the Widgets API, you can get and set properties, get and set values, and modify the interface for Wakanda's widgets.



waf.widgets.messageText.setValue(event.result);


$$('messageText').setValue(event.result);

Datasource API





You use the methods in the Datasource API to manage widgets associated with datasources on your application's Pages.



waf.sources.person.query("city == :1", inputCity + '*');

Wakanda Training


Dataprovider API





The methods of the Dataprovider API are used to manage data directly on the client.


Use this API to access data for 3rd party widgets or for custom client-side processing of data.



ds.Company.query("name = :1", qString + "*", {
   onSuccess: function(ev1) {

   }
});
